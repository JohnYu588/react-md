# Button

Buttons communicate actions that users can take. They are typically placed throughout your UI, in places like:

- Dialogs
- Modal windows
- Forms
- Cards
- Toolbars

## Simple Button

The `Button` comes in multiple variants: `flat` (default), `outlined`, `contained`
