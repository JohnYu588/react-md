# Form Message

The `FormMessage` component is used to display additional help text or error
messages and generally placed below the related form component.

## Form Level Validation Message

This component can be used to create a form-level validation message when
submitting instead of needing to display error messages with each form field.

```demo source="./FormLevelValidationMessage.tsx"

```

## Form Message Container

The `FormMessageContainer` is a simple wrapper used to conditionally render the
`FormMessage` along with any children if `messageProps` exist.

> !Info! Every form component in `react-md` uses this component behind the
> scenes already. In addition, the [useTextField](/hooks/use-text-field) and
> [useNumberField](/hooks/use-number-field) hooks will dynamically provide these
> props.
