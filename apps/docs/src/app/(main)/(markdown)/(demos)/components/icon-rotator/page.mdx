# Icon Rotator

Use the `IconRotator` component to rotate icons or other components using a
simple `transform` transition.

# Simple Example

The default behavior will be to rotate from `0deg` to `180deg` when the
`rotated` prop is `true`.

# Custom Rotation

The rotation can be changed by updating the [$SASSDOC](icon-rotate-from) and
[$SASSDOC](icon-rotate-to) Sass variables or updating the `--rmd-icon-rotate-from`
and `--rmd-icon-rotate-to` custom properties using the [$SASSDOC](icon-set-var)
mixin.
