# Theme

The global theme colors can be configured by updating the default Sass variables with
custom values. A recommended setup is creating a new `_everything.scss` file that `@forward`s
everything from `react-md` with any overrides required in the app and any additional Sass
helpers required for developing. Here is an example below showcasing the default theme variables:

```scss fileName="src/_everything.scss"
@use "@react-md/core/colors";
@forward "@react-md/core" with (
  $color-scheme: light,
  $primary-color: colors.$blue-500,
  $secondary-color: colors.$orange-a-400,
  $warning-color: colors.$deep-orange-a-400,
  $error-color: colors.$red-500,
  $success-color: colors.$green-a-700,
  $light-theme-background-color: #fafafa,
  $light-theme-surface-color: #fff,
  $light-theme-text-primary-color: color.adjust(#000, $lightness: 13%),
  $light-theme-text-secondary-color: color.adjust(#000, $lightness: 46%),
  $light-theme-text-hint-color: color.adjust(#000, $lightness: 66%),
  $light-theme-text-disabled-color: color.adjust(#000, $lightness: 62%),
  $dark-theme-background-color: #121212,
  $dark-theme-surface-color: #424242,
  $dark-theme-text-primary-color: color.adjust(#fff, $lightness: -15%),
  $dark-theme-text-secondary-color: color.adjust(#fff, $lightness: -30%),
  $dark-theme-text-hint-color: color.adjust(#fff, $lightness: -50%),
  $dark-theme-text-disabled-color: color.adjust(#fff, $lightness: -50%)
);
```

Now `@use` this file throughout the app instead of `@react-md/core` to generate styles maintaining
the overridden theme and configuration.

```scss fileName="src/index.scss"
@use "everything";

// generate all the styles
@include everything.styles;
```

See the [Color Palette](./color-palette), [Dark Mode](./dark-mode), and [Theme Builder](./theme-builder)
pages to see everything that can be configured.
